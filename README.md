# Explora Ciudades - Feeds RSS y Atom

Este repositorio contiene la implementación de feeds de sindicación para el blog **Explora Ciudades**, especializado en guías de viajes urbanos.

## Contenido
- `viajes.rss`: Feed en formato RSS 2.0.
- `viajes.atom`: Feed en formato Atom 1.0.

## Validación
Ambos archivos han sido validados mediante el **W3C Feed Validation Service*.
